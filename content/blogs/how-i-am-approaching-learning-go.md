Lately I have been very intertested in learning go as a new language, partially because I want to get into Devops and partially because of all the reviews and good thing I have heard about go.
But after watching [Sriniously](https://youtu.be/QTmck00dDoU?si=87ng4EFJaol7CIwo), my perspective changed entirely, I am coming from TS background and at every step of the way I was comparing my go learnings with TS way of doing things. 
Which was making me not appreciate the langugage as much, because honestly go is not made to replicate stuff from TypeScript. 
Go has a very different way of working, and it has its own set of strengths, so learning go with a clean slate instead of comparing it with some other language you already know will help you appreciate the language better and also not make you quit learning it.

Anyways the resources which I am using to learn go are:
1. Go Official Documentation (the best and only resource you will need to learn go).
2. Sriniously Youtube Playlist
3. Matt Kodvab youtube playlist
4. One2N go bootcamp exercises

For now I am only referring to these ones, will update them as I pick up some new ones.
